# EasyPrompt

**Build better AI prompts without starting from a blank page.**

EasyPrompt helps you browse reusable prompt assets for ChatGPT, Claude, Gemini, Codex, and other AI tools.

[**Try EasyPrompt**](https://easy-prompt-theta.vercel.app/)

## What It Is

EasyPrompt is built around three public content types:

- **Templates**: reusable forms that generate a custom prompt.
- **Prompts**: ready-to-copy instructions you can use now.
- **Workflows**: guided multi-step playbooks for larger tasks.

Use a Template when you need a custom prompt.
Use a Prompt when the instruction is already ready.
Use a Workflow when one prompt is not enough.

## Why EasyPrompt

- Browse curated Templates, Prompts, and Workflows.
- Fill in short Template fields to generate a polished prompt.
- Copy ready Prompts directly into your AI tool.
- Follow Workflows for launches, job applications, research, content planning, UX reviews, and more.
- Save, edit, and reuse your own Templates and Prompts with an account.

## Built With

Next.js, React, TypeScript, and Supabase.
